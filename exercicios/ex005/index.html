HTML5 E CSS3

O que é HTML5?

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

Para que serve HTML5?

Para apresentar e estruturar o conteúdo na web. Com o HTML5, por exemplo, é possível ver vídeos do YouTube sem o Flash.

O que é CSS3?

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets, pela qual se define estilos para um projeto web.

Para que serve CSS3?

Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.
